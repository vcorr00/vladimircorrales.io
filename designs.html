Fuel Tracker Application

This design was curated for a class project in the course Software Engineering 1. The main purpose was to create a minimalistic interface where users would be able to easily input data. The use of vibrant colors were put in place to give the user the feeling of energy, this would help in being motivated to attain your caloric goals. A login, signup, home, profile, search, and daily input pages were created to better demonstrate the flow of the application.

Zara Webpage Redesign

This redesign of the Zara webpage was curated for a class project in the course Human/Computer Interactions. The main purpose of this project was to redesign an existing webpage.The redesigned Zara webpage features a modern and user-friendly interface that enhances the online shopping experience. The homepage now showcases a clean and visually appealing layout, with a dynamic carousel highlighting the latest collections and trends. Improved navigation tools, including a persistent top menu and intuitive search bar, make it easier for users to find their desired products.
